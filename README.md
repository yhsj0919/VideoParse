# VideoParse

**该代码仅供研究学习,切勿商用**


视频解析，java-android版，目标是支持各大网站的解析，现在只做了部分网站的核心算法，不定期更新
部分算法来自 https://github.com/soimort/you-get


用法:在浏览器打开视频网站,点击分享,在系统分项找到本应用即可,


**该工具现在仅仅实现了解析算法,并没有对数据进行封装展示,这个后期会处理**
